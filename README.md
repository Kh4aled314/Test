## Suggested Git exercises

1. **Basics** ‚Äî `git status`, `git add`, `git commit`, `git log`
2. **Branches** ‚Äî Create `feature/login`, `feature/dark-mode`, merge them
3. **Conflicts** ‚Äî Edit the same line in two branches, then merge
4. **Stash** ‚Äî `git stash` and `git stash pop` while switching branches
5. **Undo** ‚Äî Practice `git restore`, `git reset`, and `git revert`
6. **Remote** ‚Äî `git push`, `git pull`, `git fetch`
7. **Pull requests** ‚Äî Open a PR on GitHub and review your own changes
8. **Tags** ‚Äî `git tag v1.0.0` and push tags to remote
